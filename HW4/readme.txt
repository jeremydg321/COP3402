1. On any command line, head to the root directory of the source codes's folder.
Do this by entering cd "filename" until you have found your root.

2. To compile the code, type "gcc lex.c parser.c vm.c compiler.c" in the command line
and enter. A file called "a.exe" will appear.

3. The name of the file with the PL/0 code is called "input.txt." To run the program, type
"a.exe input.txt" and the executable will run.
